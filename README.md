# Texas Hold'em v5 Spectacle

Render 可部署的联机德州扑克升级版。

## v5 新增/修改

- 牌型提示更干净：只显示“你开出了 X”，不再显示杂七杂八文字。
- 个人牌型提示只做小提示和光效，大型吹捧/嘲讽只在一局结束时出现。
- 不同牌型特效差异大幅增强：
  - 皇家同花顺：皇冠、金色风暴、陨石、爆闪。
  - 同花顺：霓虹隧道、激光线。
  - 四条：冲击波和光柱。
  - 葫芦：宝石坠落。
  - 同花：水波和花色雨。
  - 顺子：闪电线。
  - 三条/两对/一对：不同等级光效。
- 破产玩家不会离开房间，会继续留在桌边观战，并标记 Spectating。
- 破产玩家在一局结束时会触发红色裂屏、骷髅雨、爆炸特效。
- 赢家文案更无脑吹捧。
- 输家文案攻击性更强但仍是朋友局整活风格。
- 新增表情栏：每个人都可以发送表情，所有人都能看到浮动表情。

## Render 设置

Build Command:

```bash
npm install
```

Start Command:

```bash
npm start
```

把本项目的 `package.json`、`server.js`、`README.md` 和 `public` 文件夹上传覆盖 GitHub 仓库后，Render 会自动重新部署。
