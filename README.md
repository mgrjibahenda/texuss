# Texas Hold'em v7 Effects Fix

## 修复/增强

- 赢家结算文字变干净：只显示玩家名、牌型、赢得筹码。
- 删除赢家无脑吹捧长文字。
- 破产文字简化为“爆仓观战”，但破产特效保留并增强。
- 表情栏改为右侧固定浮动，避免在 iPad / 手机浏览器底部被挡住。
- 公共牌层级提高，避免 3 人局时被玩家座位挡住。
- 顺子特效增强为明显闪电、⚡ 图标、网格闪烁。
- 同花、葫芦、四条、同花顺、皇家同花顺等特效差异加大。
- 保留 v6 音效功能。

## Render 设置

Build Command:

```bash
npm install
```

Start Command:

```bash
npm start
```

上传覆盖 GitHub 仓库后，Render 会自动重新部署。
