# Texas Hold'em v4 Premium Plus

这是 Render 可部署的联机德州扑克升级版。

## 主要功能

- 进入界面只需要 Player name。
- 删除 Custom ID 和多余按钮。
- Dealer/房主可以在开局前给每个玩家单独设置筹码。
- Dealer/房主可以一键 Set All 设置所有玩家筹码。
- 玩家均匀分布在圆形牌桌周围。
- 2 人坐对面，3 人 120 度分开，更多玩家自动均匀分布。
- 非本人回合不显示行动按钮。
- 本人回合只显示可用操作。
- 10 不再显示为 T。
- 不同赢家牌型有不同特效。
- 高级牌型有更华丽动画。
- 赢家吹捧文案随机变化。
- 破产玩家嘲讽文案随机变化。
- 一局结束后只有 Dealer/房主能点 Next Hand。
- 当公共牌和自己的手牌组合成牌型时，会出现提示、光效和小动画。

## Render 设置

Build Command:

```bash
npm install
```

Start Command:

```bash
npm start
```

Render 免费版第一次打开可能需要等待 30-60 秒。
